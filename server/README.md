# Kukac Server

## Instalação
Para executar o projeto, é necessário ter o [Node.js](https://nodejs.org/) v10 ou superior instalada

Instale as dependências e inicie o servidor

```sh
cd kukac
cd server
npm i
npm run dev
```

Para ambientes de produção

```sh
cd kukac
cd server
npm run build
npm start
```
